# Rocky Scoop Bucket

Personal Scoop bucket for Rocky's CLI tools.

## Install
```powershell
scoop bucket add rocky https://github.com/i-rocky/bucket
```

## Apps
- [`watch`](https://github.com/i-rocky/watch)
- [`nim`](https://github.com/oyro-os/nim)

## Auto-update
This bucket includes a GitHub Actions workflow that runs `checkver.ps1` to
update manifests automatically.
